# My dbt Project

This project contains SQL models with automated documentation generation.

## Features

- ✅ Free SQL documentation generation
- ✅ Uses Ollama (open-source LLM)
- ✅ Documentation lives next to SQL files
- ✅ Zero cost

## Structure

- `models/` - dbt SQL models
- `business_docs/` - Business documentation
- `.github/workflows/` - GitHub Actions automation
